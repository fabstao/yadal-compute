# YADAL Compute Service for Libvirt/KVM

## INTRO

This is alpha version, use at your own risk.
This is a basic VM automated creation service, CLI only, for Libvirt.

## REQUIREMENTS

* Libvirt and QEMU
* virt-install, virt-resize, qemu-img
* Python 3 with pip
* bash
